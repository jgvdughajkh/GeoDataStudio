
# GeoDataStudio: 地质空间数据处理与三维可视化平台

## 项目简介

**GeoDataStudio** 是一款专为资源勘查工程设计的桌面级数据分析工具。它实现了地质监测数据（如水位、瓦斯、应力）的时间序列分析与空间三维建模的有机结合。本项目展示了如何利用高效率算法解决复杂的地质空间插值与可视化问题。

  

## 核心特性

### 多维数据解耦管理：

- **2D 监测流**：处理随时间变化的钻孔水位、瓦斯浓度及岩层应力数据 。

- **3D 空间流**：基于经纬度或坐标系的地质体空间点云处理 。

### 高性能空间建模：

- 集成 **nanoflann** 库实现 2D KD-Tree 索引，将空间查询复杂度降至 $O(\log N)$ 。

- 采用 **反距离权重插值 (IDW) 算法**，并引入平滑系数优化地形形态 。

### 异步计算架构：

- 使用 `QtConcurrent `开启多线程计算引擎，在进行百万级网格渲染时保持 UI 界面流畅响应 。

### 工程级可视化：
- 基于 `QtCharts`和 `QtDataVisualization` 实现动态折线图与三维彩色渲染地形 。

### 现代 UI 设计：
- 深色工业风界面，支持自动布局优化与高 DPI 显示 。
  
## 编译与环境

- **最低要求**：CMake 3.19+, Qt 6.5+ (REQUIRED: Core, Widgets, Charts, DataVisualization, Concurrent) 。

- **依赖库**：项目已内置单头文件高性能库 `nanoflann.hpp`。

```bash
mkdir build && cd build
cmake ..
make
```
  

## 算法科学依据

**空间插值**：采用地质统计学经典的 IDW 模型，计算公式为：$$Z_p = \frac{\sum_{i=1}^n \frac{Z_i}{d_i^2  + s^2}}{\sum_{i=1}^n \frac{1}{d_i^2  + s^2}}$$
-   $Z_p$​：待插值点的预测值
-   $Z_i​$：第 i 个已知钻孔点的实测值
-   $d_i$​：待插值点与第 i 个已知点的平面距离
-   $s$：平滑系数，软件内固定为 0.25，有效缓解插值奇点问题
-   $n$：参与插值的邻域点数，软件内固定为最近的 15 个点，兼顾插值精度与计算效率

**检索加速**：通过 KD-Tree 对离散钻孔点进行空间分割，仅选取最近的 15 个样点参与局部插值，极大地提升了大规模矿区建模的效率 。

## 项目结构

`mainwindow.cpp/h`: 核心逻辑与多线程调度 。

`nanoflann.hpp`: 空间索引算法底层库 。

`CMakeLists.txt`: 模块化构建配置 。

`mainwindow.ui`: 响应式界面布局定义 。

  

## 应用场景

矿井涌水动态监测与水位面预警。

资源勘查中离散钻孔数据的快速 DEM 生成。

矿山瓦斯与应力分布的空间趋势分析。
